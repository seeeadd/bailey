# Bailey Vann Creative AI Empire - Workshop Landing Page

## Overview

A premium, conversion-optimized landing page for Bailey Vann's free 3-day AI Etsy workshop. Designed to attract and convert 35-55 year old women (moms, corporate workers, side-hustlers) who want to build a $3-10K/month Etsy shop without becoming content creators.

## Design Philosophy

**"Quietly Premium Educator Who Delivers Real Results"**

This page intentionally moves away from typical "fem-preneur" bright colors and geometric Canva templates toward a sophisticated, organic aesthetic that builds trust through:

- Hand-drawn elements vs typical "professional" design
- Real screenshots/numbers vs vague claims
- Sage/cream/gold palette vs typical bright colors
- Imperfect/organic shapes vs geometric templates
- Authentic handwriting/signatures for personal touch

## Conversion Psychology (Jason Fladlien Framework)

### NESB Elements
- **New**: AI-powered system (not manual guessing)
- **Easy**: Without being "techy" or posting on social media
- **Safe**: Even if you've tried Etsy before and failed
- **Big**: $3-10K/month with $3 digital files

### Pattern Interrupts
- "If you've bought 10+ courses..." (calls out elephant in room)
- "quit within 45 days" (specificity builds trust)
- "quiet corner of Etsy" (creates curiosity)

### Paradigm Installation
- Shows the manual way vs. automatic way
- Data + AI + email > guessing + algorithm worship
- Positions as different from everything they've tried

### Pain Amplification
- Course fatigue acknowledged directly
- "shouting into the void" feeling
- Family asking "how's that thing you're doing?"

### Future Pacing
- Feature → benefit → life change in each outcome card
- "waking up to cha-ching notifications"
- "respect" and "freedom" not just money

## Customization Guide

### Change Workshop Date
Edit `js/main.js` line 12:
```javascript
const targetDate = new Date('2025-12-02T19:00:00-05:00').getTime();
```

### Update Color Scheme
Edit CSS custom properties in `css/styles.css`:
```css
:root {
    --sage: #7B9F8C;
    --cream: #F5F2ED;
    --gold: #D4AF37;
    /* etc. */
}
```

### Modify Spots Remaining
Edit `js/main.js` line 159 and HTML line 654:
```javascript
let spots = 147; // Starting number
```

### Connect Email Service
Edit `js/main.js` line 66-77 to add your API integration
